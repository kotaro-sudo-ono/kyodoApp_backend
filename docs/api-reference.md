# 弓道アプリ バックエンド API リファレンス

**ベースURL**: `http://localhost:8082`

## 認証

- `/user/register` および `/user/login` は認証不要（permitAll）
- その他の全エンドポイントは **JWT Bearer トークン**が必須

```
Authorization: Bearer <token>
```

---

## User（ユーザー）

### POST /user/register — ユーザー登録

新規ユーザーを登録する。

**認証**: 不要

**リクエストボディ**

| フィールド | 型     | 必須 | 説明             |
|-----------|--------|------|------------------|
| name      | String | 必須 | ユーザー名       |
| email     | String | 必須 | メールアドレス   |
| password  | String | 必須 | パスワード（平文）|

```json
{
  "name": "山田太郎",
  "email": "yamada@example.com",
  "password": "password123"
}
```

**レスポンス**

| フィールド | 型      | 説明                   |
|-----------|---------|------------------------|
| id        | String? | ユーザーID（UUID）     |
| name      | String  | ユーザー名             |
| email     | String  | メールアドレス         |

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "name": "山田太郎",
  "email": "yamada@example.com"
}
```

**ステータスコード**

| コード | 説明                         |
|--------|------------------------------|
| 200    | 登録成功                     |
| 400    | リクエスト不正               |

---

### POST /user/login — ユーザーログイン

メールアドレスとパスワードで認証し、JWTトークンを取得する。

**認証**: 不要

**リクエストボディ**

| フィールド | 型     | 必須 | 説明           |
|-----------|--------|------|----------------|
| email     | String | 必須 | メールアドレス |
| password  | String | 必須 | パスワード     |

```json
{
  "email": "yamada@example.com",
  "password": "password123"
}
```

**レスポンス**

| フィールド | 型     | 説明             |
|-----------|--------|------------------|
| token     | String | JWTアクセストークン |

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

**ステータスコード**

| コード | 説明                                   |
|--------|----------------------------------------|
| 200    | ログイン成功                           |
| 400    | リクエスト不正                         |
| 401    | 認証失敗（メールアドレス or パスワード不正）|

---

### GET /user — 全ユーザー取得

登録されている全ユーザーのリストを取得する。

**認証**: JWT必須

**リクエストボディ**: なし

**レスポンス**: ユーザーオブジェクトの配列

| フィールド | 型      | 説明                 |
|-----------|---------|----------------------|
| id        | String? | ユーザーID（UUID）   |
| name      | String  | ユーザー名           |
| email     | String  | メールアドレス       |

```json
[
  {
    "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "name": "山田太郎",
    "email": "yamada@example.com"
  },
  {
    "id": "b2c3d4e5-f6a7-8901-bcde-f12345678901",
    "name": "鈴木花子",
    "email": "suzuki@example.com"
  }
]
```

**ステータスコード**

| コード | 説明                         |
|--------|------------------------------|
| 200    | 取得成功                     |
| 401    | 認証失敗（トークンなし or 無効）|

---

### GET /user/{id} — ユーザーIDによるユーザー取得

指定したIDのユーザー情報を取得する。

**認証**: JWT必須

**パスパラメータ**

| パラメータ | 型      | 説明               |
|-----------|---------|---------------------|
| id        | String? | ユーザーID（UUID）  |

**リクエストボディ**: なし

**レスポンス**

| フィールド | 型      | 説明                 |
|-----------|---------|----------------------|
| userId    | String? | ユーザーID（UUID）   |
| name      | String  | ユーザー名           |
| email     | String  | メールアドレス       |

```json
{
  "userId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "name": "山田太郎",
  "email": "yamada@example.com"
}
```

**ステータスコード**

| コード | 説明                         |
|--------|------------------------------|
| 200    | 取得成功                     |
| 401    | 認証失敗（トークンなし or 無効）|
| 404    | 指定したユーザーが見つからない|

## BelongingGroup（所属グループ）

### POST /belongingGroup/register — グループ登録

新しい所属グループを登録する。

**認証**: JWT必須

**リクエストボディ**

| フィールド | 型      | 必須 | 説明                   |
|-----------|---------|------|------------------------|
| groupId   | String? | 任意 | グループID（サーバー生成の場合null）|
| name      | String  | 必須 | グループ名             |

```json
{
  "groupId": null,
  "name": "初段クラス"
}
```

**レスポンス**

| フィールド | 型      | 説明                   |
|-----------|---------|------------------------|
| groupId   | String? | グループID（UUID）     |
| name      | String  | グループ名             |

```json
{
  "groupId": "b2c3d4e5-f6a7-8901-bcde-f12345678901",
  "name": "初段クラス"
}
```

**ステータスコード**

| コード | 説明                         |
|--------|------------------------------|
| 200    | 登録成功                     |
| 401    | 認証失敗（トークンなし or 無効）|

---

### POST /belongingGroup/assign-group — ユーザーへのグループ割り当て

指定したユーザーに所属グループを割り当てる。

**認証**: JWT必須

**リクエストボディ**

| フィールド | 型      | 必須 | 説明               |
|-----------|---------|------|---------------------|
| groupId   | String? | 任意 | 割り当てるグループID|
| userId    | String? | 任意 | 対象ユーザーID      |

```json
{
  "groupId": "b2c3d4e5-f6a7-8901-bcde-f12345678901",
  "userId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
}
```

**レスポンス**: テキスト（text/plain）

成功時:
```
Group assigned successfully
```

失敗時:
```
Error assigning group: User not found
```

**ステータスコード**

| コード | 説明                                             |
|--------|--------------------------------------------------|
| 200    | 割り当て成功                                     |
| 400    | 割り当て失敗（ユーザーまたはグループが存在しない等）|
| 401    | 認証失敗（トークンなし or 無効）                 |

---

## Record（射撃記録）

### POST /record/save — 射撃記録の保存

ユーザーの射撃記録（的中数・総射数）を保存する。

**認証**: JWT必須

**リクエストボディ**

| フィールド  | 型     | 必須 | 説明                       |
|------------|--------|------|----------------------------|
| hitCount   | Int    | 必須 | 的中数                     |
| totalShots | Int    | 必須 | 総射数                     |
| userId     | String | 必須 | 記録を保存するユーザーID   |

```json
{
  "hitCount": 7,
  "totalShots": 10,
  "userId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
}
```

**レスポンス**

| フィールド  | 型      | 説明                   |
|------------|---------|------------------------|
| recordId   | String? | 記録ID（UUID）         |
| hitCount   | Int     | 的中数                 |
| totalShots | Int     | 総射数                 |

```json
{
  "recordId": "c3d4e5f6-a7b8-9012-cdef-123456789012",
  "hitCount": 7,
  "totalShots": 10
}
```

**ステータスコード**

| コード | 説明                         |
|--------|------------------------------|
| 200    | 保存成功                     |
| 400    | リクエスト不正               |
| 401    | 認証失敗（トークンなし or 無効）|

---

### GET /record/user/{userId} — ユーザーIDによる射撃記録一覧取得

指定したユーザーの全射撃記録を取得する。

**認証**: JWT必須

**パスパラメータ**

| パラメータ | 型     | 説明               |
|-----------|--------|---------------------|
| userId    | String | ユーザーID（UUID）  |

**リクエストボディ**: なし

**レスポンス**: 記録オブジェクトの配列

| フィールド  | 型      | 説明           |
|------------|---------|----------------|
| recordId   | String? | 記録ID（UUID） |
| hitCount   | Int     | 的中数         |
| totalShots | Int     | 総射数         |

```json
[
  {
    "recordId": "c3d4e5f6-a7b8-9012-cdef-123456789012",
    "hitCount": 7,
    "totalShots": 10
  },
  {
    "recordId": "d4e5f6a7-b8c9-0123-def0-234567890123",
    "hitCount": 5,
    "totalShots": 8
  }
]
```

**ステータスコード**

| コード | 説明                         |
|--------|------------------------------|
| 200    | 取得成功                     |
| 401    | 認証失敗（トークンなし or 無効）|
